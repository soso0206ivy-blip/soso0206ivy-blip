
## 流程判斷

### if .. else

```javascript
if (指定條件1) {
  陳述式 1;
} else if (指定條件 2) {
  陳述式 2;
} else if (指定條件 n) {
  陳述式 n;
} else {
  最後陳述式;
}
```

```javascript
function checkData() {
  var input = prompt("請輸入你的名字");
  if (input.length > 3) {
    alert("你輸入的名字大於 3 個字");
  } else {
    alert("你輸入的名字小於等於 3 個字");
  }
}

```

### switch

```javascript
switch (運算式) {
  case 標籤 1:
    陳述式 1
    [break;]
  case 標籤 2:
    陳述式 2
    [break;]
    ...
  default:
    陳述式
    [break;]
}
```


