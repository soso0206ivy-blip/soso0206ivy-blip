
## DOM

### 取得物件 (getElementById)

```htmlmixed
<div id="firstDiv"></div>
```

```javascript
var ele = document.getElementById('firstDiv');
ele.textContent = 'Hello JavaScript!!'; // 修改內容

```

### 新增物件 (createElement, appendChild)

```htmlmixed
<div id="firstDiv"></div>
```
```javascript
var parent = document.getElementById('firstDiv');
var newEle = document.createElement('div');

newEle.id = 'secondDiv';
newEle.style.backgroundColor = '#ffeeee';
newEle.style.fontSize = '20px';
newEle.textContent = '這是新的物件';

parent.appendChild(newEle);

```

### 移除物件 (removeChild)

```htmlmixed
<div id="firstDiv"><div id="secondDiv">
我是第二個物件
</div></div>
```
```javascript
var parent = document.getElementById('firstDiv');
var child = document.getElementById('secondDiv');
parent.removeChild(child);

```

### 事件練習

```htmlmixed
<!DOCTYPE html>
<html>
<head>
  <title>DOM 物件操作</title>
  <style>
    .box {
      width: 200px;
      height: 200px;
      background-color: #f0f0f0;
      border: 1px solid #ccc;
      text-align: center;
      line-height: 200px;
    }
  </style>
</head>
<body>
  <div class="box" id="firstBox">點擊我！</div>
</body>
</html>

```

```javascript
var box = document.getElementById('firstBox');
box.addEventListener('click', function() {
  box.textContent = '被點擊了！';
  box.style.backgroundColor = 'red';
});

```
