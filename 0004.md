
## 宣告

一般來說 JavaScript 不需要經過變數宣告即可直接使用變數，除非有做嚴謹模式的宣告。

```javascript
'use strict'
```

### 宣告方式

宣告變數有三種方式

- var
宣告可隨意更改其內容的變數
- let
宣告可隨意更改其內容的區塊區域變數
- const
宣告常數

經過宣告但未賦予值的變數可用 undefined 進行判斷和排查
```javascript
var userName;
if (userName === undefined) {
  alert("尚未定義名稱");
} else {
  alert(userName + " 很高興認識你❤️");
}
```
