
## 常數

屬於固定不可變的值例如：PI

```javascript
pi = 3.14159265359
```

## 變數

JavaScript 的變數或是函數大小寫有區分，這點需要注意。

### 命名

- 必須使用字母、下底線、錢字號作為開頭
- 除了開頭自原有規定以外其餘可使用 A-Z / a-z / 0-9 / _ 等字元組成

#### 命名方式

##### 慣用的命名方式

- 駝峰式（Camel Case）
    - 大駝峰（同 Pascal Case）：每個單字的首字均大寫
        ```
        UserName
        ContactAddress
        ```
    - 小駝峰：第一個單字的首字小寫，其餘首字大寫
        ```
        userName
        contactAddress
        ```

- 蛇型（Snake Case）：單字與單字間使用 `_` 連結
    ```
    user_name
    contact_address
    ```

還有其他像是：
- Kebab Case `user-name`
- 匈牙利命名法（Hungarian notation） `strUserName`

同一個專案中命名方式需要統一避免各種命名方式混用造成閱讀困擾。

```javascript
// 正確且大多數使用的方式
var userName = "JerryHU";
var user_name = "JerryHU";

// 可以使用
var _userName = "JerryHU";
var 我的名字 = "JerryHU";

// 錯誤
var 2000userName = "JerryHU";
var #userName = "JerryHU";  // 不能 $ 或是 _ 以外的符號
var var = "JerryHU"; // var 是保留字

// 被視為兩個不同變數
var userName = "JerryHU";
var username = "JerryHU";

// 可以使用，但命名應該儘量具有意義
var qwert = "JerryHU";
var aaa123 = "JerryHU";

```

### 變數範圍

var 與 let 的區別

```javascript
if (true){
    var userName = "JerryHU";
}
alert(userName);
```

```javascript
if (true){
    let userName = "JerryHU";
}
alert(userName);
```

### 資料型別

基本型別

- boolean : true / false
- null
- undefined : 未給值
- number : 888 / 3.14159
- string : "JerryHU"
- symbol : ES2015 開始才有

以及物件 (object)，Array、Json 都是物件的一種

### 動態型別

同一個變數允許被賦予不同的資料型別

```javascript
var userAccount = 888888888;
userAccount = "JerryHU 的帳戶有 888,888,888 元";
alert(userAccount);
```

型別會依照運算子不同而進行不同的型別處理

- 數值自動轉字串
    ```javascript
    var myAccount1 = "我的帳戶有 " + 888888888
    alert(myAccount1);
    var myAccount2 = 888888888 + " 在我的帳戶裡"
    alert(myAccount2);
    ```

- 字串自動轉數值
    ```javascript
    alert("17888" - 222)
    alert("17" + 888)
    ```

### 運算子

#### 比較運算子

- 等於 ==
- 不等於 !=
- 大於 >
- 小於 <
- 大於等於 >=
- 小於等於 <=

比較年紀是否超過 30

```javascript
var age = 29;
if (age > 30){
    alert("有超過 30 歲");
}
```

#### 邏輯運算子

- 且 &&
- 或 ||
- 否(NOT) !

假設 var x = 14 是現在的時間，判斷 x 是否是下午的語法參考：
```javascript
x > 12 && x < 18
```


#### 算數運算子

- 一般運算 : + - * /
```javascript
var x = 17666 + 222
var x = 17888 - 222
var x = 666 * 88
var x = 666 / 22
```
- 取餘數 : %
```javascript
var x = 17888 % 33
```
- 加一 ++
- 減一 --
